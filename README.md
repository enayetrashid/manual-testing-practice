# Manual Testing Practice — ATM Simulator

## Overview

This repository demonstrates structured manual testing on a real-world Python ATM application using standard QA documentation and traceability practices.

The system under test is the ATM Simulator from Repo 5.

👉 Application Repository:  
https://github.com/enayetrashid/atm-simulator

---

## Objective

To validate core banking operations and demonstrate a professional QA workflow through:

- Requirement analysis
- Test planning
- Test scenario design
- Test case writing
- Bug reporting
- Requirement traceability

---

## Scope

### In Scope
- User authentication
- Balance enquiry
- Deposit
- Withdrawal
- Fund transfer
- Transaction history
- Basic persistence validation

### Out of Scope
- Security testing
- Performance benchmarking beyond basic checks
- Multi-user concurrency testing
- GUI and accessibility testing
