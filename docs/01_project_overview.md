# Project Overview — ATM Simulator Manual Testing

## Purpose

This repository documents manual testing performed against the ATM Simulator application.

## System Under Test

The application under test is a CLI-based ATM Simulator developed in a separate repository:

👉 https://github.com/enayetrashid/atm-simulator

## Main Functions Covered

- User authentication
- Balance enquiry
- Deposit
- Withdrawal
- Fund transfer
- Transaction history

## Testing Goal

The goal is to validate expected behavior, identify defects, and demonstrate a structured QA workflow using requirements, test scenarios, test cases, bug reports, and traceability.

## Notes

The requirements and test cases in this repository were refined and aligned with the actual ATM implementation to ensure accurate coverage and realistic validation.
